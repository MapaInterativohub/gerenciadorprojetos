- **nomeDoProjeto**: Nome oficial do projeto.

- **alunosParticipantesDoProjeto**: Lista com os nomes dos alunos envolvidos.
- **linkGit**: Link para o repositório GitHub ou outra plataforma de versionamento.
- **linkImage**: Imagem representativa do projeto.
